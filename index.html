COMO FUNCIONA O ÍNDICE

Dados financeiros anônimos

Analisamos os dados das PMEs ativas no Sistema Omie com faturamento anual de até R$ 50.000.000, de maneira 100% anônima, todos divididos por setores da economia. A privacidade é garantida pela Lei Geral de Proteção de Dados Pessoais (LGPD).

Setorização por CNAE

Analisamos as atividades econômicas (setores de atuação) de cada empresa e as segmentamos por CNAE, permitindo a avaliação dos principais setores de atuação das PMEs brasileiras.

Remoção do efeito inflacionário

Corrigimos os dados das movimentações financeiras por segmento, com um cálculo específico para retirar a inflação e comparar os dados sem considerar a evolução de preços, a fim de obter o crescimento real de cada setor.
